# ​​ Gerenciamento de Estoque

Projeto em Python focado em criar uma aplicação para **controle básico de estoque de produtos**, com funcionalidades fundamentais que servem como base para versões futuras com interface ou relatórios mais avançados.



##  Tecnologias utilizadas

- **Python 3**
- Possível uso de bibliotecas como **Pandas**, **Tkinter** ou **Openpyxl** para manipulação de dados e interface, dependendo da versão atual.



##  Funcionalidades

- Estrutura inicial em Python para gerenciamento de produtos (cadastro, edição, remoção).
- Armazenamento dos dados em arquivo (CSV, Excel ou similar).
- Interface de terminal (ou gráfica, caso esteja implementada) para operar os registros.
